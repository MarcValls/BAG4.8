# Generación de Imágenes

## Objetivo

Convertir intención visual en un prompt reproducible.

## Qué debe salir de aquí

- prompt base
- variantes controladas
- restricciones claras
- criterios de validación
- notas de fallos y ajustes

## Regla práctica

Si no puedes explicar por qué una variante existe, todavía no está lista para
volver a usarla.
