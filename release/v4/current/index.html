Gestor de instalaciones

Detectá las instalaciones BAGO en tu máquina, inspeccionalas y ejecutá acciones sobre cada una. La web no toca tu PC: corré un comando, pegá el resultado, y operá desde acá con privacidad total.

Cómo usar este gestor

  1. En tu terminal (PowerShell o cmd), corré: bago list-installs --plain
  2. Copiá todo el output (un JSON largo en una línea).
  3. Click Pegar output arriba. La web parsea el JSON y muestra las instalaciones detectadas.
  4. Click en cualquier card para expandirla. Cada acción es un comando listo para copiar.
Aviso de ejecución: en la app Electron los botones Ejecutar abren una ventana PowerShell visible con el comando. No se ejecuta nada oculto ni silencioso. En la web solo se copian comandos.
