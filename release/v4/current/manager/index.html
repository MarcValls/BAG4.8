BAGO · Gestor de Instalaciones

standalone · offline · v—

Gestiona todas tus instalaciones BAGO

Controla instalaciones, roles BAGO, piezas, compatibilidad y evidencia. Las descargas de GitHub son solo una accion mas del gestor, no el centro de la app.

Cómo usarlo como gestor

  1. Revisa primero Node Control: registry, piezas, matriz y conectores.
  2. Comprueba roles BAGO y las instalaciones detectadas.
  3. Usa releases solo para instalar, actualizar o probar una version aparte.
  4. En Electron, las acciones largas se ejecutan con progreso en el manager; en navegador, se copian comandos.
Aviso de ejecución: el manager no abre PowerShell visible por defecto. Las instalaciones van a un job con progreso y los comandos interactivos se copian para ejecución manual.
Android: para móvil usa el gestor mínimo en manager/android/index.html (providers no-locales).
