# BAGO 1.0-RC5 revisada (RC5-R1)

Este paquete sustituye la propuesta RC5 anterior y mantiene la sustitución íntegra de RC1, RC2, RC3 y RC4.

## Contenido

- `CANON_BAGO_1.0-RC5_REVISADA.md`: canon completo revisado.
- `CANON_BAGO_1.0-RC5_REVISADA.pdf`: versión maquetada.
- `RC5_REVISADA_CHANGELOG.md`: cambios.
- `RC5_REVISADA_TRACEABILITY.md`: trazabilidad.
- `AUDITORIA_4_ISSUES_RESOLUTION.md`: cierre normativo de las cuatro incidencias.
- `PREGUNTA_ESPEJO_SPEC.md`: especificación matemática y contractual.
- `contracts/ui/v1/`: seis contratos JSON Schema nuevos o revisados.
- `tests/RC5_R1_TEST_MATRIX.md`: matriz de pruebas añadidas.
- `sources/pregunta_espejo_canva.pptx`: fuente conceptual aportada.
- `MANIFEST.json`: hashes SHA-256 del paquete.

La existencia de estos documentos no demuestra implementación. Define las puertas que el código deberá superar.
