    chat                Inicia el REPL de chat

    exec                Ejecuta un comando slash sin abrir el REPL
    validate            Gate real de validacion: security, contratos, culpas,
                        claims, providers
    profiles            Muestra el mapa de stable/des/ign y el flujo
                        recomendado
    install             Instala/repara BAGO desde la copia local, sin descarga
    uninstall           Desinstala BAGO de la ruta indicada
    claim               Claim Evidence Ledger -- afirmaciones trazables
    config              Gestiona configuracion
    llm                 Gestiona arranque provider-aware
    engine              Estado del backend avanzado bago_true
    appdata             Estado de instalacion AppData BAGO
    cmd-rl              Estado del puente AppData cmd-rl/Spiral
    rl                  RL shadow bridge
    serve               Inicia servidor API HTTP
    evidence            Genera bundle de evidencias verificables
    monitor             Monitor HTML en tiempo real de procesos BAGO
    orchestrate         Orchestrator v4 -- Flujo Operativo (Regla Fundamental)
    issues              Flujo rapido de issues (list/take/close)
    scan                Herramientas de analisis portables (secrets, deps,
                        todos, tokens, dead, names, sincerity, net, metrics,
                        infra, heal, security, doctor, commit, git, all)
    canary              Honeytokens -- trampas de deteccion de intrusos
    backup              Backups del proyecto con rotacion
    project             Gestiona la estructura portable .bago del proyecto
    preflight           Ejecuta checks de preflight portables
    toolsmith           Gestiona toolboxes de agentes
    issues-gh           Gestiona issues del repositorio
    agent               Gestiona spiral agents
    guard               Guardian de deuda tecnica -- previene patrones antes
                        de commitear
    route               Gestion de presets de routing y contrato activo
    inventory           Cataloga capacidades del proyecto
    list-installs       Escanea el sistema e imprime JSON con todas las
                        instalaciones BAGO (para el gestor de la landing)
    install-role        Elige que copia BAGO se usa como active/dev/launch
    node                Gestor de registry, policy, evidence y compatibilidad
